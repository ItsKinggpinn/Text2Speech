# Text2Speech

This is a Text to speech Converter made with HTML, CSS, JavaScript and Tailwind CSS that simply runs on Web Speech API.

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Web Speech API
